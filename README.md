# 🚀 Brunel Cyber Security Society

> *"Curious about how systems break — and how to defend them."*

Welcome to the official GitHub of the **Brunel Cyber Security Society**. We're a student-led community at Brunel University London, building the next generation of cyber defenders.

---

## 🔐 What We Do

| Activity | Description |
| :--- | :--- |
| **Workshops** | Hands-on sessions covering web exploitation, forensics, cryptography, and more |
| **CTF Competitions** | Capture The Flag events – beginners welcome! |
| **Socials** | Hack nights, networking, and community building |
